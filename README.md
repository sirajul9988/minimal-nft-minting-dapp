# Minimal NFT Minting DApp

A high-performance, beginner-friendly frontend template for interacting with NFT smart contracts. This repository demonstrates how to connect a browser wallet, read blockchain state, and execute minting transactions.

## Features
- **Wallet Integration**: Seamless connection using Ethers.js.
- **Real-time State**: Automatically updates supply and balance after minting.
- **Provider Agnostic**: Works with MetaMask, Coinbase Wallet, and other injected providers.
- **Flat Structure**: All logic contained in a single directory for easy deployment.

## Technical Stack
- React.js
- Ethers.js (v6)
- CSS3 (Standard Modules)

## Getting Started
1. Update `CONTRACT_ADDRESS` in `App.js`.
2. Install dependencies: `npm install ethers react`.
3. Start the development server: `npm start`.
